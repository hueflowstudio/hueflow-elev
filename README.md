AutoCAD에서 평면도 → 4면 전개도를 자동으로 펼쳐주는 LISP 도구입니다.
인테리어 설계 실무에서 매번 반복되던 작업을 한 번에 처리하려고 만들었어요.

휴플로우 스튜디오에서 만들었습니다.

---

## 뭐가 되나요

`HF` 명령어 하나로 방의 4면 전개도 외곽선을 펼쳐서 그려줍니다.

- 평면 폴리라인 선택 → 자동으로 4면 분리 → 일렬로 펼침
- 깔끔한 외곽선(왁구) 버전 — 마감/디멘션은 직접 작업하기 좋게 비워둠
- 평소에 30분 걸리던 전개도 외곽 잡기가 1분이면 끝

---

## 설치

1. 위 **Code** 버튼 → **Download ZIP** 으로 받기
   (또는 우측 **Releases** 에서 최신 버전 다운로드)
2. 압축 풀고 `HUEFLOW_ELEV2.lsp` 파일을 작업 폴더에 두기
3. AutoCAD에서 `APPLOAD` 명령어 실행 → 파일 선택 → **Load**
4. 명령행에 `HF` 입력하면 시작

> 💡 매번 불러오기 귀찮으면 APPLOAD 창의 **Startup Suite(시작 세트)** 에 등록해두세요. 캐드 켤 때마다 자동으로 로드돼요.

---

## 사용법

1. `HF` 입력
2. 방 평면 폴리라인 클릭
3. 천장 높이 입력
4. 전개도 배치할 위치 클릭

끝.
